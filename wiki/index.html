General Commands

  • b!help → get help menu.
  • b!info → get bot information.
  • b!ping → get bot current ping.
  • b!serverinfo → get server information.
  • b!userinfo → get user information.
  • b!suggest → Suggest something to get added to the bot.
  • b!ug → Report a bug found on the bot.

Music Commands

  • b!play → play music through url or search term.
  • b!pause → pause the current audio.
  • b!resume → resume the paused audio.
  • b!stop → stop the current audio & clear queue.
  • b!skip → skip the current audio.
  • b!leave → leave the voice channel & clear queue.
  • b!loop → loop through the queue.
  • b!queue → check the current queue list.
  • b!np → check what's currently playing.
  • b!volume → set the audio volume.
  • b!remove → remove a listing in the queue.
  • b!clear → clear the queue.
